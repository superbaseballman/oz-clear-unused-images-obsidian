# i18n 翻译说明

本插件支持多语言国际化 (i18n)，目前支持以下语言：

- **英文 (en)** - 默认语言
- **简体中文 (zh-cn)** 
- **繁体中文 (zh-tw)**

## 语言检测

插件会自动检测 Obsidian 的系统语言设置：
- 如果系统语言是简体中文，自动使用简体中文翻译
- 如果系统语言是繁体中文或香港中文，自动使用繁体中文翻译
- 其他情况使用英文翻译

## 添加新语言

如果你想添加新的语言支持，请按照以下步骤操作：

### 1. 在 `src/i18n.ts` 中添加新的翻译对象

```typescript
// 例如：添加日语翻译
const jaTranslations: Partial<I18nTranslations> = {
    'close': '閉じる',
    'cancel': 'キャンセル',
    'delete': '削除',
    // ... 其他翻译
};
```

### 2. 更新 translationsMap

```typescript
const translationsMap: Record<Language, Partial<I18nTranslations>> = {
    'en': enTranslations,
    'zh-cn': zhCNTranslations,
    'zh-tw': zhTWTranslations,
    'ja': jaTranslations, // 添加新语言
};
```

### 3. 更新 Language 类型

```typescript
export type Language = 'en' | 'zh-cn' | 'zh-tw' | 'ja';
```

### 4. 更新 detectLanguage 方法

```typescript
private detectLanguage() {
    const systemLang = Platform.getLocale();
    
    if (systemLang === 'ja') {
        this.currentLanguage = 'ja';
        this.translations = jaTranslations;
    }
    // ... 其他语言判断
}
```

## 翻译键列表

### 通用
- `close` - 关闭
- `cancel` - 取消
- `delete` - 删除
- `ignore` - 忽略
- `confirm` - 确认

### 命令名称
- `command.clear.images` - 清除未使用的图片
- `command.clear.attachments` - 清除未使用的附件

### 通知消息
- `notice.all.images.used` - 所有图片都被使用或在排除文件夹中
- `notice.all.attachments.used` - 所有附件都被使用或在排除文件夹中
- `notice.file.ignored` - 已忽略文件
- `notice.file.already.ignored` - 文件已在忽略列表中
- `notice.plugin.not.found` - 无法找到插件实例
- `notice.error.ignoring.file` - 忽略文件时出错
- `notice.deleted.success` - 成功删除文件
- `notice.no.files.selected` - 没有选择文件
- `notice.plugin.reload.required` - 需要重新加载插件

### 设置界面
- `settings.title` - 设置标题
- `settings.ribbonIcon.name` - 功能区图标
- `settings.ribbonIcon.desc` - 功能区图标描述
- `settings.logsModal.name` - 删除日志
- `settings.logsModal.desc` - 删除日志描述
- `settings.deleteOption.name` - 删除目标位置
- `settings.deleteOption.desc` - 删除目标位置描述
- `settings.deleteOption.permanent` - 永久删除
- `settings.deleteOption.obsidian.trash` - 移动到 Obsidian 回收站
- `settings.deleteOption.system.trash` - 移动到系统回收站
- `settings.excludedFolders.name` - 排除的文件夹
- `settings.excludedFolders.desc` - 排除的文件夹描述
- `settings.excludedFiles.name` - 排除的文件
- `settings.excludedFiles.desc` - 排除的文件描述
- `settings.excludeSubfolders.name` - 排除子文件夹
- `settings.excludeSubfolders.desc` - 排除子文件夹描述

### 模态框
- `modal.select.files.to.delete` - 选择要删除的文件
- `modal.delete.selected` - 删除所选文件
- `modal.select.all` - 选择所有文件
- `modal.deselect.all` - 取消选择
- `modal.ignore.file` - 忽略此文件
- `modal.file.path.click.hint` - 点击打开文件

### 删除日志
- `logs.moved.obsidian.trash` - 移动到 Obsidian 回收站
- `logs.moved.system.trash` - 移动到系统回收站
- `logs.deleted.permanently` - 永久删除

## 参数化翻译

某些翻译文本支持参数替换，使用 `{参数名}` 格式：

```typescript
// 示例
i18n.t('notice.file.ignored', { file: filePath });
// 输出：已忽略文件：path/to/file.png

i18n.t('notice.deleted.success', { count: '5' });
// 输出：成功删除 5 个文件。
```
